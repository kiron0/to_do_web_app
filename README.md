## Installation

```bash
git clone https://github.com/kiron0/to_do_web_app.git
```

```bash
cd to_do_web_app
```

```bash
yarn
```

```bash
code .
```

```bash
yarn start
```

## Prerequisites

The following open source packages are used in this project:

- React JS v18
- TailwindCSS
- DaisyUI
- React Router DOM v6
- React Icons
